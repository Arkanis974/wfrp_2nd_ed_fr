# Points de Folie (PF)
**Source :** Livre de base p.18 / Core Rulebook p.18 *Insanity Points (IP)*
**Profil :** Profil secondaire
## 📜 Description
Cette caractéristique représente la statut de la santé mentale de votre personnage. Les PJ débutent avec 0 point de Folie, mais ils peuvent en acquérir en cours de partie à la suite d'expériences traumatisantes ou de souffrances particulièrement insupportables. Votre MJ vous en dira davantage sur cet aspect du jeu.

La page "[[Caractéristiques/Les caractéristiques\|Les caractéristiques]]" indique comment sont déterminées chacune des caractéristiques de votre personnage en fonction de sa race.
