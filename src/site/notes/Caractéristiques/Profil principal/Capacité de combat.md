# Capacité de combat (CC)
**Source :** Livre de base p.17 / Core Rulebook p.17 *Weapon skill (WS)*
**Profil :** Profil principal

## 📜 Description
Cette caractéristique représente l'aptitude de votre personnage en combat rapproché, aussi bien armé qu'à mains nues.

La page "[[Caractéristiques/Les caractéristiques\|Les caractéristiques]]" indique comment sont déterminées chacune des caractéristiques de votre personnage en fonction de sa race.
