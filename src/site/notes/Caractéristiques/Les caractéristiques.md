# Les caractéristiques
Les caractéristiques définissent votre personnage par ses capacités brutes dans un certain nombres de domaines physiques et mentaux.
Elles sont divisées en deux groupes : le *profil principal* et le *profil secondaire*. Les caractéristiques du *profil principal* affichent une valeur allant de 0 à 100. Plus la valeur est élevée plus votre personnage est doué dans ce domaine. Les caractéristiques du *profil secondaire* vont généralement de 0 à 10. De manière générale, les valeurs les plus élevées sont les meilleures mais cela peut varier dans certains cas (pour les [[Caractéristiques/Profil secondaire/Point de Folie\|points de folie]] notamment).

Les caractéristiques du *profil principal* sont les suivantes : [[Caractéristiques/Profil principal/Agilité\|Agilité]], [[Caractéristiques/Profil principal/Capacité de combat\|Capacité de combat]], [[Caractéristiques/Profil principal/Capacité de tir\|Capacité de tir]], [[Caractéristiques/Profil principal/Endurance\|Endurance]], [[Caractéristiques/Profil principal/Force\|Force]], [[Caractéristiques/Profil principal/Force mentale\|Force mentale]], [[Caractéristiques/Profil principal/Intelligence\|Intelligence]], [[Caractéristiques/Profil principal/Sociabilité\|Sociabilité]].

Les caractéristique du *profil secondaire* sont les suivantes : [[Caractéristiques/Profil secondaire/Attaques\|Attaques]], [[Caractéristiques/Profil secondaire/Blessures\|Blessures]], [[Caractéristiques/Profil secondaire/Bonus d'Endurance\|Bonus d'Endurance]], [[Caractéristiques/Profil secondaire/Bonus d'Endurance\|Bonus d'Endurance]], [[Caractéristiques/Profil secondaire/Magie\|Magie]], [[Caractéristiques/Profil secondaire/Mouvement\|Mouvement]], [[Caractéristiques/Profil secondaire/Point de Destin\|Point de Destin]], [[Caractéristiques/Profil secondaire/Point de Folie\|Point de Folie]].
