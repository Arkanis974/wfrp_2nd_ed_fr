# Kossar kislévite
**Type :** Carrière de base
**Source :** Livre de base p.47 / Core Rulebook p.42 *Kislevite kossar*
## 📜 Description
Les Kossars sont originaires d’une tribu ungol qui vivait jadis dans le nord-est de l’[[Empire\|Empire]]. Un peuple oriental, les Gospodars, envahit la région, soumit les Ungols et fonda la nation de [[Kislev\|Kislev]]. Durant ce conflit, les Kossars louèrent leurs services de mercenaires aux Gospodars et combattirent les autres tribus ungols. Leur style de combat sans pareil impressionna grandement la noblesse gospodar et depuis cette époque, des régiments de Kossars servent les tzars de [[Kislev\|Kislev]]. Aujourd’hui, les Kossars ne constituent plus une tribu, mais une vaste unité militaire étendue sur l’ensemble de [[Kislev\|Kislev]]. Ces hommes sont armés d’arcs et de grandes haches, ce qui les rend très polyvalents sur le champ de bataille. Las des guerres incessantes qui agitent leur patrie, nombre de Kossars se sont rendus dans l’[[Empire\|Empire]] pour y mener une vie de mercenaires ou d’aventuriers.

*Note : s’il s’agit de votre carrière de départ, vous êtes originaire de Kislev.*
