# Projectile puissant
**Source :** Livre de base p.100 / Core Rulebook p.99 *Mighty missile*

## 📜 Description
Vous êtes capable d'infliger un maximum de dégâts à l'aide de vos projectiles magiques. Vous bénéficiez d'un bonus de *+1* aux jets de dégâts des sorts de projectiles magiques.
