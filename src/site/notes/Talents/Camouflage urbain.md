# Camouflage urbain
**Source :** Livre de base p.97/ Core Rulebook p.96 *Alley cat*

## 📜 Description
Vous êtes à l'aise dans la rue comme un chat de gouttière. Vous bénéficiez d'un bonus de *+10%* aux tests de [[Compétences/Déplacement silencieux\|Déplacement silencieux]] et de [[Compétences/Dissimulation\|Dissimulation]] en milieu urbain.
