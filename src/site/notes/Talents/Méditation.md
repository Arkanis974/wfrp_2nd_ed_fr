# Méditation
**Source :** Livre de base p.100 / Core Rulebook p.99 *Meditation*

## 📜 Description
Vous pouvez vous concentrer et ignorer le monde qui vous entoure. Quand vous pratiquez *la magie rituelle*, vous bénéficiez au jet d'incantation d'un bonus égal à votre caractéristique de [[Caractéristiques/Profil secondaire/Magie\|Magie]].
