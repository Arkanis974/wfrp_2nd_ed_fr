# Tir en puissance
**Source :** Livre de base p.102 / Core Rulebook p.99 *Mighty shot*

## 📜 Description
Vous êtes capable d'infliger un maximum de dégâts à l'aide de vos tirs. Vous bénéficiez d'un bonus de *+1 aux jets de dégâts* des armes de tirs.
