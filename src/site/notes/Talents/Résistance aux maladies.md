# Résistance aux maladies
**Source :** Livre de base p.101 / Core Rulebook p.100 *Resistance to disease*

## 📜 Description
Votre constitution vous immunise contre la plupart des maladies. Vous bénéficiez d'un bonus de *+10%* aux tests d'[[Caractéristiques/Profil principal/Endurance\|Endurance]] destinés à résister aux maladies.
