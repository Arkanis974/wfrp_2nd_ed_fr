# Parade éclair
**Source :** Livre de base p.100 / Core Rulebook p.99 *Lightning parry*

## 📜 Description
Quand vous effectuez une attaque rapide (cf. *Chapitre 6 : Combat, dégâts et mouvements*), vous pouvez renoncer à l'une de vos attaques pour bénéficier d'une parade gratuite. Si vous avez 3 en [[Caractéristiques/Profil secondaire/Attaques\|Attaques]], par exemple, vous avez droit à deux attaques et une parade avec cette attaque rapide. La limite d'une parade par round doit cependant toujours être respectée.
