# Sixième sens
**Source :** Livre de base p.101 / Core Rulebook p.100 *Sixth sens*

## 📜 Description
Vous êtes parcouru d'une étrange sensation quand vous êtes en danger, un avertissement qui vous sauve parfois la mise avant que le pire n'advienne. Quand le danger est proche, le MJ effectue un test de [[Caractéristiques/Profil principal/Force mentale\|Force mentale]] secret. En cas de succès, il peut vous dire que vous avez un mauvais pressentiment ou que vous vous sentez observé. Ce talent peut vous éviter d'être pris par surprise tandis que tous vos alliés le seront.
