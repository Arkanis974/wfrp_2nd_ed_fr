# Sain d'esprit
**Source :** Livre de base p.101 / Core Rulebook p.101 *Strong minded*

## 📜 Description
Votre détermination est votre rempart contre les chocs psychologiques. Vous n'avez pas besoin d'effectuer de test de *Folie* avant d'avoir *8 [[Caractéristiques/Profil secondaire/Point de Folie\|Points de Folie (PF)]]* et vous n'en gagnez pas automatiquement une avant d'avoir *14 [[Caractéristiques/Profil secondaire/Point de Folie\|PF]]*.
