# Acrobatie équestre
**Source :** Livre de base p.96 / Core Rulebook p.102 *Trick riding*

## 📜 Description
Vous êtes capable de prouesses quand vous êtes à cheval. Vous pouvez faire de l'équilibre sur une monture lancée au galop, sauter d'un cheval à un autre, etc. Dans les circonstances les plus extrêmes, il vous suffit de réussir un test d'[[Compétences/Équitation\|Équitation]], et même alors, vous bénéficier d'un bonus de +10%.
