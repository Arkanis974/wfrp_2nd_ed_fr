# Talent artistique
**Source :** Livre de base p.102 / Core Rulebook p.97 *Artistic*

## 📜 Description
Vous avez une âme d'artiste. Vous bénéficiez d'un bonus de *+20%* au tests de [[Compétences/Métier\|Métier]] (artiste) et aux tests d'[[Compétences/Evaluation\|Évaluation]] concernant les oeuvres d'arts.
