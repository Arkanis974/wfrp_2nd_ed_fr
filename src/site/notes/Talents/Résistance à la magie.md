# Résistance à la magie
**Source :** Livre de base p.100 / Core Rulebook p.100 *Resistance to magic*

## 📜 Description
Vous disposez d'une résistance naturelle contre le pouvoir de la magie. Vous bénéficiez d'un bonus de *+10%* aux tests de [[Caractéristiques/Profil principal/Force mentale\|Force mentale]] visant à résister à la magie.
