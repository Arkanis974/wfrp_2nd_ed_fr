# Imitation
**Source :** Livre de base p.98 / Core Rulebook p.99 *Mimic*

## 📜 Description
Vous avez une bonne oreille en ce qui concerne les voix et les accents, et vous savez les reproduire fidèlement. Vous bénéficiez d'un bonus de *+10%* aux tests de [[Compétences/Déguisement\|Déguisement]] si vous devez imiter une voix dans le but de vous faire passer pour autrui, aux tests d'[[Compétences/Expression artistique\|Expression artistique]] (acteur, bouffon, clown, comédien et conteur) et aux tests de [[Compétences/Langue\|Langue]] quand vous essayer de faire croire que vous êtes originaires d'un autre pays en parlant la langue.
