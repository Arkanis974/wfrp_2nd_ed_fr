# Tir de précision
**Source :** Livre de base p.102 / Core Rulebook p.101 *Sure shot*

## 📜 Description
Vous savez trouver la faille dans l'armure de vos ennemis. Quand vous réussissez une attaque à distance, vous pouvez *ignorer 1 point d'armure*. Si votre cible n'a pas d'armure, ce talent n'a aucun effet.
