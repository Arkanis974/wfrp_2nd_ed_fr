# Ambidextre
**Source :** Livre de base p.97 / Core Rulebook p.97 *Ambidextrous*

## 📜 Description
Vous savez parfaitement vous servir de vos deux mains. Vous ne subissez pas le malus de -20% en [[Caractéristiques/Profil principal/Capacité de combat\|CC]] ou en [[Caractéristiques/Profil principal/Capacité de tir\|CT]] quand vous utilisez une arme avec votre main non directrice.
