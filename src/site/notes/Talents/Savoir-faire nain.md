# Savoir-faire nain
**Source :** Livre de base p.101 / Core Rulebook p.98 *Dwarfcraft*

## 📜 Description
Les membres de votre race sont des artisans nés. Vous bénéficiez d'un bonus de *+10%* aux tests pratiqués conjointement avec les compétences de [[Compétences/Métier\|Métier]] suivantes : 
- arquebusier
- brasseur
- cristallier
- fabricant d'armes
- fabricant d'armures
- forgeron
- maçon
- mineur.
