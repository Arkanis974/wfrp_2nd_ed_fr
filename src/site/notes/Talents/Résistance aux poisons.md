# Résistance aux poisons
**Source :** Livre de base p.101 / Core Rulebook p.100 *Resistance to poison*

## 📜 Description
Votre robustesse vous permet de résister aux effets de nombreux poisons. Vous bénéficiez d'un bonus de *+10%* aux tests d'[[Caractéristiques/Profil principal/Endurance\|Endurance]] destinés à résister aux poisons.
