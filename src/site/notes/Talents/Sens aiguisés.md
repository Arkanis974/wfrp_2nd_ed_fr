# Sens aiguisés
**Source :** Livre de base p.101 / Core Rulebook p.99 *Keen senses*

## 📜 Description
Vos sens sont naturellement aiguisés. Vous bénéficiez d'un bonus de *+20%* aux tests de [[Compétences/Perception\|Perception]].
