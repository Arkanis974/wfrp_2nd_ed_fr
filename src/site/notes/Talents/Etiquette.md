# Étiquette
**Source :** Livre de base p.98 / Core Rulebook p.98 *Etiquette*

## 📜 Description
Vous connaissez bien les mœurs de la haute société. Vous bénéficiez d'un bonus de *+10%* aux tests de [[Compétences/Charisme\|Charisme]] et de [[Compétences/Commérage\|Commérage]] quand vous êtes face à des membre de la noblesse. Ce bonus s'applique également à toutes les situation où la connaissances de l'étiquette est importante (comme se faire passer pour un noble avec la compétence [[Compétences/Déguisement\|Déguisement]], par exemple).
