# Intrigant
**Source :** Livre de base p.98 / Core Rulebook p.100 *Schemer*

## 📜 Description
Vous êtes passé maitre dans l'art des intrigues personnelles. Vous bénéficiez d'un bonus de *+10%* aux tests de [[Compétences/Charisme\|Charisme]] liés aux intrigues et aux tests de [[Caractéristiques/Profil principal/Force mentale\|Force mentale]] destinés à résister au [[Compétences/Charisme\|Charisme]] d'autrui.
