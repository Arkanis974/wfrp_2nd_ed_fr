# Troublant
**Source :** Livre de base p.102 / Core Rulebook p.102 *Unsettling*

## 📜 Description
Votre simple présence dérange vos adversaires. Vos ennemis doivent réussir un test de [[Caractéristiques/Profil principal/Force mentale\|Force mentale]] quand ils vous voient pour éviter de subir un malus de *-10%* en [[Caractéristiques/Profil principal/Capacité de combat\|Capacité de combat]] et en [[Caractéristiques/Profil principal/Capacité de tir\|Capacité de tir]]. Ils peuvent retenter ce test à chaque round jusqu'à ce qu'ils le réussissent ou que vous soyez hors de vue.
