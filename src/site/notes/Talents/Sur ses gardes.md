# Sur ses gardes
**Source :** Livre de base p.102 / Core Rulebook p.100 *Quick draw*

## 📜 Description
Vos réflexes aiguisés vous permettent de dégainer rapidement vos armes et d'autres objets. Une fois par round, vous pouvez utiliser l'action *dégainer* en tant qu'*action gratuite*.
