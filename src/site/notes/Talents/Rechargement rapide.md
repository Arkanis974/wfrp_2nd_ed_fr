# Rechargement rapide
**Source :** Livre de base p.100 / Core Rulebook p.100 *Rapid reload*

## 📜 Description
Vous rechargez vos armes à distance avec aisance. Le temps de rechargement des armes de tir est réduit d'une demi-action en ce qui vous concerne. Par exemple, vous pouvez recharger une arbalète au prix d'une demi-action, alors qu'il faut normalement une action complète pour y parvenir. Si le temps de rechargement de l'arme est déjà d'une demi-action, il s'agit désormais d'une action action gratuite en ce qui vous concerne. Vous rechargez de telles armes presque instantanément, ce qui vous permet de faire des attaques rapides grâce à elles. Pour plus de détails concernant les attaques rapides, reportez-vous au *Chapitre 6 : Combat, dégats et mouvements*.
