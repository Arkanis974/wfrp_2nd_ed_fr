# Menaçant
**Source :** Livre de base p.100 / Core Rulebook p.99 *Menacing*

## 📜 Description
Votre taille, votre aspect ou votre comportement vous confèrent une présence imposante. Vous bénéficiez d'un bonus de *+10%* aux tests d'[[Compétences/Intimidation\|Intimidation]] et de [[Compétences/Torture\|Torture]].
