# Maitre artilleur
**Source :** Livre de base p.100 / Core Rulebook p.99 *Master gunner*

## 📜 Description
Pour vous, le temps de rechargement des armes à poudre est réduit d'une demi-action. Si vous disposez aussi du talent [[Talents/Rechargement rapide\|Rechargement rapide]], cet avantage est cumulable (ce qui réduit le temps de rechargement des armes à poudre d'une action complète).
