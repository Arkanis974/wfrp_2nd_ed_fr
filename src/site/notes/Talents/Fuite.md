# Fuite
**Source :** Livre de base p.98 / Core Rulebook p.98 *Flee!*

## 📜 Description
Quand votre vie est en jeu, vous êtes capable de pointes de vitesse incroyables. Quand vous fuyez un combat ou un grave danger, vous bénéficiez d'un bonus de *+1* en [[Caractéristiques/Profil secondaire/Mouvement\|Mouvement]] pendant *1d10 rounds*.
