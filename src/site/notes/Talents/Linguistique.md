# Linguistique
**Source :** Livre de base p.99 / Core Rulebook p.99 *Linguistics*

## 📜 Description
Vous avez une affinité particulière pour les langues étrangères. Vous bénéficiez d'un bonus de *+10%* aux tests de [[Compétences/Lire-Écrire\|Lire/Écrire]] et de [[Compétences/Langue\|Langue]].
