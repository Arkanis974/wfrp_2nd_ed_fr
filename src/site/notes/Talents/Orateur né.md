# Orateur né
**Source :** Livre de base p.100 / Core Rulebook p.99 *Master orator*

## 📜 Description
Vous maitrisez si bien l'art oratoire que vous pouvez enflammer des foules entières. Le nombre de personnes que vous pouvez affecter à l'aide de la compétence [[Compétences/Charisme\|Charisme]] est multiplié par 100. Vous devez disposer du talent [[Talents/Eloquence\|Éloquence]] pour avoir droit à celui-ci.
