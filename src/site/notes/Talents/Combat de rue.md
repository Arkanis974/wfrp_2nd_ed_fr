# Combat de rue
**Source :** Livre de base p.97 / Core Rulebook p.101 *Street fighting*

## 📜 Description
Vous avez appris à vous battre dans la rue. Vos attaques à mains nues bénéficient d'un bonus de *+10%* en [[Caractéristiques/Profil principal/Capacité de combat\|Capacité de combat]]. Par ailleurs, vous bénéficiez d'un bonus de *+1* aux jets de *dégâts* de telles attaques.
