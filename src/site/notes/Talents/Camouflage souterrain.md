# Camouflage souterrain
**Source :** Livre de base p.97/ Core Rulebook p.102 *Tunnel rat*

## 📜 Description
Les profondeurs de la terre sont votre second foyer. Vous bénéficiez d'un bonus de *+10%* aux tests de [[Compétences/Déplacement silencieux\|Déplacement silencieux]] et de [[Compétences/Dissimulation\|Dissimulation]] dans les environnement souterrains.
