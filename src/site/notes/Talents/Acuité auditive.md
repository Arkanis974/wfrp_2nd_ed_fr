# Acuité auditive
**Source :** Livre de base p.96 / Core Rulebook p.96 *Acute hearing*

## 📜 Description
Vous disposez d'une ouïe exceptionnelle. Vous bénéficiez d'un bonus de 20% aux tests de [[Perception\|Perception]] impliquant l'ouïe.
