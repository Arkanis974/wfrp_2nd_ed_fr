# Robuste
**Source :** Livre de base p.101 / Core Rulebook p.101 *Sturdy*

## 📜 Description
Vous êtes musclé et résistant. Vous ne subissez pas de malus en [[Caractéristiques/Profil secondaire/Mouvement\|Mouvement]] quand vous portez une armure de plaques complètes (armure lourde). Pour plus d'informations concernant les armures, reportez vous au *Chapitre 5 : L'équipement*.
