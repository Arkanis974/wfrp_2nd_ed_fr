# Camouflage rural
**Source :** Livre de base p.97/ Core Rulebook p.100 *Rover*

## 📜 Description
Vous êtes parfaitement dans votre élément dans la nature. Vous bénéficiez d'un bonus de *+10%* aux tests de [[Compétences/Déplacement silencieux\|Déplacement silencieux]] et de [[Compétences/Dissimulation\|Dissimulation]] en milieu rural.
