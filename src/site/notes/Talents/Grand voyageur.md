# Grand voyageur
**Source :** Livre de base p.98 / Core Rulebook p.100 *Seasoned traveller*

## 📜 Description
Vos longs voyages ont façonné votre expérience. Vous bénéficiez d'un bonus de *+10%* aux tests de [[Compétences/Connaissances générales\|Connaissances générales]] et de [[Compétences/Langue\|Langue]].
