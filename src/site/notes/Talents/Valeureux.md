# Valeureux
**Source :** Livre de base p.102 / Core Rulebook p.101 *Stout-hearted*

## 📜 Description
Vous êtes d'une bravoure exceptionnelle. Vous bénéficiez d'un bonus de *+10%* aux tests de[[Peur\|Peur]] et de [[Terreur\|Terreur]], ainsi qu'aux tests de [[Caractéristiques/Profil principal/Force mentale\|Force mentale]] visant à résister à une tentative d'[[Compétences/Intimidation\|Intimidation]].
