# Code de la rue
**Source :** Livre de base p.97 / Core Rulebook p.101 *Streetwise*

## 📜 Description
Vous savez comment vous conduire dans la rue. Vous bénéficiez d'un bonus de *+10%* aux tests de [[Compétences/Charisme\|Charisme]] et [[Compétences/Commérage\|Commérage]] dans tous vos rapports avec le milieu du crime.
