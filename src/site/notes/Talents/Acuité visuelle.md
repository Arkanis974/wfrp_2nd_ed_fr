# Acuité visuelle
**Source :** Livre de base p.96 / Core Rulebook p.98 *Excellent vision*

## 📜 Description
Vous avez des yeux de lynx. Vous bénéficiez d'un bonus de +10% aux tests de [[Perception\|Perception]] liés à la vue et aux tests de [[Compétences/Lecture sur les lèvres\|Lecture sur les lèvres]].
