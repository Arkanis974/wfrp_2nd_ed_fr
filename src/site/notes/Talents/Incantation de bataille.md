# Incantation de bataille
**Source :** Livre de base p.98 / Core Rulebook p.97 *Armoured casting*

## 📜 Description
Vos prières vous aident à lancer des sorts facilement même si vous portez une armure. Le malus au jet d'incantation est *réduit de 3 points* quand vous lancez des sorts divins tout en portant une armure. Par exemple, vous subissez habituellement un malus *-3* quand vous portez un [[Gilet de mailles\|gilet de maille]] mais avec le talent Incantation de bataille, ce malus passe à 0.
