# Lutte
**Source :** Livre de base p.99 / Core Rulebook p.102 *Wrestling*

## 📜 Description
Le pugilat et les prises n'ont pas de secret pour vous. Vous pouvez porter des attaques à mains nues avec un bonus de *+10%* en [[Caractéristiques/Profil principal/Capacité de combat\|Capacité de combat]] quand vous tentez une prise. De plus, vous bénéficiez d'un bonus de *+10%* à vos tests de [[Caractéristiques/Profil principal/Force\|Force]] liés à une prise.
