# Natation
**Source :** Livre de base p.95 / Core Rulebook p.96 *Swim*
**Type :** Compétence de base
**Caractéristique :** [[Caractéristiques/Profil principal/Force\|Force]]
**Talents liés :** Aucun
## 📜 Description
Cette compétence permet de nager et de plonger. Nager dans des conditions normales ne requiert aucun test de compétence. Le MJ peut en demander un quand l'eau est particulièrement agitée ou quand il s'agit de nager pendant une longue période. La caractéristique de [[Caractéristiques/Profil secondaire/Mouvement\|Mouvement]] du personnage est divisée par deux (arrondir à l'entier supérieur) quand il nage.
