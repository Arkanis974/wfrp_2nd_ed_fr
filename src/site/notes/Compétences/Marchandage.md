# Marchandage
**Source :** Livre de base p.95 / Core Rulebook p.93 *Haggle*
**Type :** Compétence de base
**Caractéristique :** [[Caractéristiques/Profil principal/Sociabilité\|Sociabilité]]
**Talents liés :** [[Talents/Dur en affaires\|Dur en affaires]]
## 📜 Description
On utilise cette compétence pour négocier des prix et des échanges. Dans le cadre de la vie quotidienne, pour négocier les prix des denrées au marché par exemple, un simple test de compétence suffit. Toutefois, dans le cas d'échanges importants (comme la négociation du prix d'un manuscrit rare), le MJ peut exiger des tests de compétences opposés, utilisant le Marchandage des deux parties en présence.
