# Déguisement
**Source :** Livre de base p.92 / Core Rulebook p.92 *Disguise*
**Type :** Compétence de base
**Caractéristique :** [[Caractéristiques/Profil principal/Sociabilité\|Sociabilité]]
**Talents liés :** [[Imitation\|Imitation]]
## 📜 Description
On utilise cette compétence pour masquer sa véritable apparence. Vêtements, accessoires et maquillages sont souvent la clef d'un déguisement réussi. Il est également possible de se faire passer pour un membre du sexe opposé, d'une autre race ou pour une personne précise, mais la tâche est bien plus ardue. Le test de Déguisement est souvent opposé à un test de [[Perception\|Perception]] de la part de l'adversaire.
