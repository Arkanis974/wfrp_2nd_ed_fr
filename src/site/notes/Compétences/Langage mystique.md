# Langage mystique
**Source :** Livre de base p.94 / Core Rulebook p.95 *Speak arcane language*
**Type :** Compétence avancée
**Caractéristique :** [[Caractéristiques/Profil principal/Intelligence\|Intelligence]]
**Talents liés :** Aucun
## 📜 Description
On utilise cette compétence pour lancer des sorts. Votre personnage doit parler un langage mystique s'il veut utiliser [[La magie\|la magie]].Contrairement aux autres langues, les langages mystiques ne servent pas à tenir une conversation, mais à manipuler des énergies magiques. Tous les parchemins et les grimoires sont rédigés en langage mystique. A l'instar de [[Compétences/Connaissances académiques\|Connaissances académiques]], Langage mystique n'est pas une compétence unique mais multiple. Les langages mystiques les plus répandus sont les suivants : le demonik (aussi appelée *lingua daemonica* par les magisters impériaux), l'elfique mystique et le maegick (aussi appelée *lingua praestentia*).
