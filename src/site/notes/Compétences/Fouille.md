# Fouille
**Source :** Livre de base p.94 / Core Rulebook p.95 *Search*
**Type :** Compétence de base
**Caractéristique :** [[Caractéristiques/Profil principal/Intelligence\|Intelligence]]
**Talents liés :** Aucun
## 📜 Description
On utilise cette compétence pour rechercher des indices, des trésors et autres objets cachés (y compris des pièges) dans un lieu donné. Un test est nécessaire pour chaque pièce ou zone.
