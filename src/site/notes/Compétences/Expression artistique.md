# Expression artistique
**Source :** Livre de base p.93 / Core Rulebook p.94 *Performer*
**Type :** Compétence avancée
**Caractéristique :** [[Caractéristiques/Profil principal/Sociabilité\|Sociabilité]]
**Talents liés :** [[Talents/Contorsionniste\|Contorsionniste]], [[Imitation\|Imitation]]
## 📜 Description
On utilise cette compétence pour divertir un groupe de spectateurs. Comme [[Compétences/Connaissances académiques\|Connaissances académiques]], Expression artistique est une compétence multiple. Les domaines les plus répandus pour cette compétence sont : acrobate, acteur, bouffon, chanteur, chiromancien, clown, comédien, conteur, cracheur de feu, danseur, jongleur, mime et musicien.
