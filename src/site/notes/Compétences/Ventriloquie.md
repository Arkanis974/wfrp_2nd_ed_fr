# Ventriloquie
**Source :** Livre de base p.96 / Core Rulebook p.96 *Ventriloquism*
**Type :** Compétence avancée
**Caractéristique :** [[Caractéristiques/Profil principal/Sociabilité\|Sociabilité]]
**Talents liés :** Aucun
## 📜 Description
Cette compétence permet de parler sans ouvrir les lèvres. Des observateurs particulièrement vigilants auront droit à un test de [[Compétences/Perception\|Perception]] opposé, à la discrétion du MJ.
