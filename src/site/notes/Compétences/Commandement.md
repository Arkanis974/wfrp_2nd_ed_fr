# Commandement
**Source :** Livre de base p.92 / Core Rulebook p.92
**Type :** Compétence de base
**Caractéristique :** [[Sociabilité\|Sociabilité]]
**Talents liés :** Aucun
## 📜 Description
On utilise cette compétence pour amener des subalternes à exécuter des ordres. Quand le test de compétence est couronné de succès, les ordres sont exécutés tels qu'ils ont été donnés. En cas d'échec, les ordres sont mal interprétés ou les subalternes ne font rien du tout (au choix du MJ). La compétence Commandement ne permet de pas de donner des ordres à de parfaits étrangers, mais uniquement à ceux qui sont déjà placés sous l'autorité de son utilisateur.
