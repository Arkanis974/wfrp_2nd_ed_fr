# Équitation
**Source :** Livre de base p.93 / Core Rulebook p.95 *Ride*
**Type :** Compétence de base
**Caractéristique :** [[Caractéristiques/Profil principal/Agilité\|Agilité]]
**Talents liés :** [[Talents/Acrobatie équestre\|Acrobatie équestre]]
## 📜 Description
Cette compétence est utilisée pour monter des chevaux et autres montures similaires. En temps normal, l'équitation ne nécessite pas de test de compétence. Cependant, le MJ peut exiger des tests de compétence pour galoper, faire la course, guider sa monture sur un terrain accidenté, sauter sur une monture en mouvement, etc.
