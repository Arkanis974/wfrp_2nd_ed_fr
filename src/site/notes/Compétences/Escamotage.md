# Escamotage
**Source :** Livre de base p.93 / Core Rulebook p.95 *Sleight of hand*
**Type :** Compétence avancée
**Caractéristique :** [[Caractéristiques/Profil principal/Agilité\|Agilité]]
**Talents liés :** Aucun
## 📜 Description
On utilise cette compétence pour faire disparaitre de petits objets dans la paume de la main, pour faire les poches de quelqu'un ou réaliser des tours de passe-passe avec des pièces ou des cartes. Le test d'Escamotage est souvent opposé à un test de [[Perception\|Perception]] de l'observateur.
