# Résistance à l'alcool
**Source :** Livre de base p.95 / Core Rulebook p.92 *Consume alcohol*
**Type :** Compétence de base
**Caractéristique :** [[Caractéristiques/Profil principal/Endurance\|Endurance]]
**Talents liés :** Aucun
## 📜 Description
On utilise cette compétence pour résister pour résister aux effets de l'alcool. Les buveurs expérimentés peuvent développer une tolérance remarquable. On effectue un test de compétence après chaque verre. Pour plus de détails concernant l'alcool et ses effets, reportez-vous au *Chapitre 5 : L'équipement*.
