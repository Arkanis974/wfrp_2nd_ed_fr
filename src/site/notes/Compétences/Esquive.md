# Esquive
**Source :** Livre de base p.93 / Core Rulebook p.92 *Dodge blow*
**Type :** Compétence avancée
**Caractéristique :** [[Caractéristiques/Profil principal/Agilité\|Agilité]]
**Talents liés :** Aucun
## 📜 Description
Cette compétence permet d'éviter les attaques au combat au corps à corps. On peut utiliser l'Esquive une fois par [[Round\|round]].
